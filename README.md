### Concepcts 
* Git: A piece of software designed for version control it tracks changes to files overtime, keeping detailed history. Its helps users avoid manual,confusing file versioning.
* Github: A web service that runs GIt software on servers,allowing users to manage projects online. Github enables collaboration and social ineraction around shared projects,sometimes called socia coding.
* Repository (Repo): A project containter that holds multiple files and their history.
* Commit: A "save" operation within Git that records changes with an optionl messages. Each commit has a unique identifier called a commit hash.
