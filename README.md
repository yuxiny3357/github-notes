### Concepcts 
* Git: A piece of software designed for version control it tracks changes to files over time, keeping detailed history. It helps users avoid manual,confusing file versioning.
* Github: A web service that runs GIt software on servers,allowing users to manage projects online. Github enables collaboration and social interaction around shared projects,sometimes called social coding.
* Repository (Repo): A project container that holds multiple files and their history.
* Commit: A "save" operation within Git that records changes with optional messages. Each commit has a unique identifier called a commit hash.
